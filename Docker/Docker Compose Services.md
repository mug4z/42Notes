[[Docker compose]]

A service is an abstract definition of a computing resource within an application which can be scaled or replaced independently from other components

It begin with a `service` keyword.
> [!example] Example
> ```yaml
> services:
  > # A services name 
 > Linux:
 >   # Image name 
 >   image: testcompose
 >   build: ./
  > ```
